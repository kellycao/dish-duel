# 开饭馆方向 - 讨论记录

> 2025-05-29 讨论整理

---

## 一、整体结构：潜水员戴夫式

**确认模板**：潜水员戴夫（Dave the Diver）

- 白天：城市探索（主要gameplay时间）
- 晚上：回饭馆经营（轻量但有操作的经营阶段）
- 两者严格交替
- 探索占大头，经营是较短的"消化/展示"阶段

---

## 二、探索部分

**参考**：天国拯救2（Kingdom Come: Deliverance 2）

- 跑图 + 对话 + 解谜 + 有动作成分
- 开放世界天津城市，故事/任务驱动
- 每个区域有NPC、故事线、分支
- 玩家在天津街头巷尾的沉浸式体验

**收集体验参考**：老头环（Elden Ring）

- 食材和菜谱散落在城市各处，每个都独特、有名字有故事
- 不是"面粉×10"消耗品，而是"天津老面发酵法""独流老醋"这样的一次性解锁
- 有的在路边能发现，有的藏在隐秘巷子里，有的需要完成NPC任务
- 获得后立刻解锁新的经营选项
- 玩家之间会交流"你找到那个XX了吗"

---

## 三、白天/晚上节奏

**白天结束方式**：以玩家主动为主，混合时间流逝

- 玩家自己决定什么时候回饭馆"开始营业"
- 白天有时间流逝，某些事件/NPC只在特定时段出现
- 可以选择不回去，但有轻微代价（如第二天顾客评价降低）
- 白天做不完的任务第二天继续——不焦虑

---

## 四、探索与经营的咬合

**食材模式：纯解锁型**

- 食材 = 一次性解锁，探索到了就永远会做这道菜
- 不存在消耗/补货压力——避免探索变成"打工买菜"
- 经营的日常变化来自**需求端**，不来自供给端

**需求端驱动经营决策：**

- 解锁了20道菜，但每晚只能上桌6道
- 每晚客群不同（工人多→硬菜，文人多→精致，小孩多→甜食）
- 特殊客人事件（美食评论家、大胃王比赛、挑剔老太太）
- 季节/天气影响需求（冬天汤好卖、夏天凉菜好卖）
- 每晚决策 = "从我的菜单里选哪几道应对今天的情况"

**探索产出不止食材：**

- 新菜品（扩展菜单）
- 食谱改良（升级已有菜品）
- 厨具（改变烹饪方式）
- 装修/扩建材料
- 人脉（新的食客群体）

## 五、探索玩法方向

### 5.1 战斗定位

**结论：KCD2式——战斗是探索中的一种遭遇，不是核心循环。**

- 探索的目的是获取食材/菜谱/人脉，不是打架
- 如果战斗是核心（老头环），"找菜"会变成战斗的奖励包装，饭馆经营感觉像副业
- 战斗不需要硬核，但需要有辨识度——"只有这个游戏里有"
- 所有遭遇/玩法的辨识度应该来自天津城市内容本身

### 5.2 天津特色内容 → 玩法素材

从天津内容出发（不从玩法模板出发），筛选后有潜力的方向：

| 内容 | 交互类型 | 类比KCD2 |
|------|---------|---------|
| 海河跳水 | 姿态/角度/时机——表演打分 | 射箭比赛 |
| 摊/吃煎饼果子 | 摊面→打蛋→刷酱→卷——操作序列 | 打铁/炼金 |
| 古文化街鉴宝 | 观察→判断→出价——眼力+博弈 | 开锁/扒窃 |
| 抖空竹 | 节奏/幅度控制——手感操作 | 磨刀 |
| 手艺（吹糖人/泥人张/风筝魏） | 塑形/制作——产出实物 | 打铁 |
| 捧哏 | 接话时机→选词→观众反应——节奏博弈 | 掷骰子(社交版) |

### 5.3 有深度的玩法方向（重点发展）

**鉴宝 — 知识积累型**

- 深度来自"玩家自己在学东西"——每件物品是一道新题
- 越玩越懂，但永远有看走眼的可能
- 对标：Papers Please、Obra Dinn（信息判断类）
- 与饭馆天然接口：鉴食材品质、老器具、菜谱真伪
- 适合做可重复的核心玩法

**捧哏 — 系统级对话设计**

- 不做成独立小游戏，而是融入所有对话——玩家在对话中始终是"接话的人"
- 核心：NPC抛话头，玩家选择怎么接、什么时候接
- 要避免燕云十六声话术系统的问题（换皮石头剪刀布，与内容脱节）
- 可能的规则方向：
  - **时机型**：在正确的节拍点插嘴，太早打断笑点，太晚冷场（接近音游）
  - **选词型**：捧/拆/转/垫四种接法，不同NPC吃不同的接法，导向不同走向
  - **能量管理型**：控制"气氛值"在目标区间——套话让对方飘，砍价适度冷
- 风险：文字内容越重要→越难系统化→越容易变成选择题；系统越抽象→越容易变成换皮数值游戏。需要原型验证

**天津话解谜 — 语言推理型**

- 玩家遇到天津方言，需要从上下文、环境线索推断意思，据此完成任务
- 对标：Chants of Sennaar（巴别塔圣歌）——纯语言推理撑起整个游戏
- 优势：
  - 天津话本身有极强喜剧性和辨识度（嘛呢、崴泥、倍儿棒、齁儿咸）
  - 不需要玩家真会天津话——推断过程就是乐趣
  - 跟饭馆天然相关（老师傅教菜、菜市场交易）
  - 解出来后"原来如此！"+ 真学了一句天津话
- 与捧哏可共存：捧哏是对话的**节奏层**，天津话是对话的**内容层**

### 5.4 轻量点缀型（不需要太深，偶尔玩有新鲜感）

- 摊煎饼果子（操作序列，做几次就会了）
- 海河跳水（表演打分，短时刺激）
- 抖空竹（手感操作，技巧展示）
- 手艺制作（产出装饰/礼物/商品）

## 六、经营玩法：上菜解谜

### 6.1 定位

经营阶段 = **上菜解谜**，不是对抗。复用菜品对决的combo/连击系统，但去掉"双方互伤"结构。

### 6.2 基本结构

- **一桌一桌服务**，不是同时应对多桌的手忙脚乱调度
- 客人坐下，有口味偏好/点菜要求
- 玩家选择上什么菜、按什么顺序上
- 连续上同口味的菜 → 满意度倍率 ×2→×3（复用combo系统）
- 套餐凑齐 → 大满足

### 6.3 压力来源（不是对抗，而是约束）

- **点菜约束**：客人有明确偏好（"要辣的""不吃甜"），限制你的combo链选择
- **目标压力**：满意度门槛与叙事能力挂钩（见七、叙事框架）——喂得越好，超自然能力触发越深

### 6.4 与对抗版（菜品对决）的区别

- 没有"对方也在伤害我"的环节
- 压力不来自敌人，来自客人需求和满意度目标
- 节奏更从容，思考空间更大——适合经营阶段"消化/展示"的定位

## 七、叙事框架：食物作为手段

### 7.1 核心洞察

如果游戏只围绕"吃"，玩家身份局限在厨子/探店博主/店主，想象空间窄。需要魔幻设定让食物成为**达成更高目标的工具**，不是目标本身。

### 7.2 食物从"目的"变"手段"后的循环结构

```
探索调查线索 → 需要接触某个人 → 把他喂开心
    → 触发超自然能力 → 获得新线索/能力
    → 引出下一个要调查的人/地方
```

### 7.3 三个候选框架（均通过初筛，待深入选择）

**A. 食忆 — 吃开心了能进入记忆**

- 能力：当食客吃到真正触动他的菜，你能进入他的记忆片段
- 菜越对味、满意度越高 → 记忆越深层
- 饭馆 = 系统性"读人"的方式
- 探索内容：在记忆世界里调查线索，不同人的记忆 = 不同区域
- 优势：天津话解谜天然成立（老人记忆里全是方言）；鉴宝可发生在记忆中的古文化街；捧哏是跟记忆人物打交道的方式
- 老天津的记忆里能看到已经消失的街巷——历史感

**B. 食链 — 食物是侦探工具**

- 能力：超自然味觉——吃过某人做的菜能感知他的情绪/意图，给别人吃你做的菜能影响他的状态（放松戒备、激发回忆、暴露破绽）
- 玩家身份：侦探，饭馆是审讯室
- 探索内容：白天跟踪线索、接触嫌疑人；晚上把关键人物约到饭馆，用精心设计的菜单"撬开"他们
- 优势：经营变成"设局"——不是随便喂饱，是针对特定人物设计菜单。每桌客人是一次情报任务

**D. 食约 — 共食结盟，双向影响**

- 能力：与人共食建立超自然纽带"食约"。食约越深（菜越好、越对味），纽带越强
- **核心亮点：双向影响——不只是我影响食客，食客也改变我**
- 你能借用对方的能力/知识/人脉/感知
- 三种借用形式（可共存）：
  - **借技能**：锁匠吃饱→你临时能开锁；书法家吃饱→你能读古文。"明天要去古文化街鉴宝，今晚得把古董贩子伺候好"
  - **借感官**：盲人茶师→超自然听觉；老渔民→闻到暗流；相声演员→读懂微表情和话外音。同一场景用不同感官探索能发现不同东西
  - **借身份/气质**：每人带有"味道印记"，喂饱后印记沾到你身上。带码头工人印记→河边的人信任你；带老学究印记→古文化街店主愿意跟你聊。本质是"社会伪装"系统，但不是换衣服，是换气质
- 探索内容：找到关键人物，搞清楚口味偏好，获取能打动他们的食材/菜谱
- 经营 = "织网"，探索 = "找线"
- 优势：经营选择直接决定明天的探索能力；社交网络可视化天然好玩；每个NPC有明确"攻略"目标
